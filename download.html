Qt : クロスプラットフォームアプリケーションフレームワークをダウンロード

  • Jump to:
  • Qt libraries
  • Qt Creator
  • Visual Studio Add-in

Qt SDKは一回インストールするだけでデスクトップ、組み込み機器やモバイルのアプリケーションを開発する際に必要なツールが含まれています。 そしてQt SDKをインストールすることはQtで開発をはじめる際で最も最適な方法です。 最新のSDKはQt SDKを最新に保つためのアップデーターが含まれており、常に最新バージョンに保つことが可能です。

8月11日にリリースされたQt SDK version 1.2.1 は次のものが含まれています

  • Qt libraries version 4.8.1
  • Simulator for Symbian phones and the Nokia N9
  • Qt Creator IDE version 2.4.1
  • Qt Mobility version 1.2
  • Qt development tools
  • Remote compilers

Windows:

  • オンラインインストーラー - 15 MB
  • オフラインインストーラー - 1.7 GB

Linux/X11 32-bit:

  • オンラインインストーラー - 26 MB
  • オフラインインストーラー - 764 MB

Linux/X11 64-bit:

  • オンラインインストーラー - 26 MB
  • オフラインインストーラーr - 767 MB

Mac OS X 10.6 or later 64-bit:

  • オンラインインストーラー - 12 MB
  • オフラインインストーラー - 760 MB

Nokia DeveloperでChecksumは確認できます。

注意: Qt 4.6.3 と Qt 4.7.3の開発者
Qt SDK 1.2は最新のSymbianプラットフォームのアプリケーションを開発するターゲットのみ含まれています。
もし Symbian が使用する Qt 4.6.3 か Symbian 3とSymbian Anna が使用する Qt 4.7.3 で開発したい場合は Qt SDK 1.1.2を利用して下さい。
一度 Qt SDK 1.1.2 場合、最近のターゲットとApp TRKを使うことを有効にして下さい。必要な機能が失われるので決してSDKアップデートを使わないで下さい。

そして Qt SDK 1.1.2 と Qt SDK 1.2.1 を同時に使いたい場合は別のユーザーアカウントにインストールする必要があることに注意して下さい。

Qt と QtCreator が あなたのプラットフォームで対応しているか確認して下さい、そしてSDKドキュメンテーションの"Getting Started"を読んで下さい。

SDK 必要要件
64ビット Apple Mac OS X 10.6以降 XCode
32,64ビット Ubuntu Linux 8.04以降
32,64ビット Microsoft Windows XP Service Pack 2, Windows Vista, or Windows 7

これらは最新の Qtライブラリです。このエディションは開発環境が導入されたOSに対応するものをダウンロードする必要があります。

ソースコードは zip (261 MB) 又は tar.gz (223 MB)で利用できます。.必要な場合はリポジトリ( qt.gitorious.org/qt )にアクセスして下さい。

  • Qtライブラリ 4.8.3 for Linux/X11 (223 MB)

  • Qtライブラリ 4.8.3 for embedded Linux (223 MB)

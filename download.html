Qtは GPL v3, LGPL v2 そして 商用版ライセンスのもとで頒布されています。ライセンスについて更に知りたい場合は ここを見て下さい。

Qt 商用版 ライセンス
Qt 商用版ライセンスはサポート、アップデート などを含む様々なライセンス条件を提供しています。 Qt 商用版ライセンスは競争上の優位性を確保し、 開発投資を保護するためや、プロプライエタリ又は商用にQtでソフトウェアを 開発する際の適切なライセンスです。

  • Qt商用版ライセンスを評価する (qt.digia.com)

Qt 5.0

下のリストからOSを参考にして、最新のQt5.0を入手してください。以下のパッケージは Qt 5.0.1 ライブラリと Qt Creator 2.7.0 を含んでいます。

  • Qt 5.0.1 - Linux 64ビット (410 MB)
  • Qt 5.0.1 - Linux 32ビット (411 MB)
  • Qt 5.0.1 - Mac (398 MB)
  • Qt 5.0.1 - Windows 32ビット (VS 2010 406 MB)

Qt5.0 のソースコードは zip (270 MB) または tar.gz (219 MB) として入手可能です。または、 ここ のその他のインストーラーもチェックしてください。 qt.gitorious.org/qt のリポジトリにもアクセスしてみてください。

旧バージョン

Qt 4.7 やその他の Qt の旧バージョンは アーカイブ で入手可能です。

Qt がシステムのプラットフォームに対応されているかを確認し、 Qt Documentation にあるインストレーションの注意を読んでください。

Qt Creator 2.7.0 は Qt 5.0 バイナリリリースに含まれています。もしスタンドアロンのインストーラーが欲しい場合は、 下のリストからOSを参考にして、最新のQt Creatorを入手してください

  • Qt Creator 2.7.0 - Windows (52 MB)
  • Qt Creator 2.7.0 - Linux/X11 32ビット (61 MB)
  • Qt Creator 2.7.0 - Linux/X11 64ビット (60 MB)
  • Qt Creator 2.7.0 - Mac (Intel 32/64ビット) (51 MB)

Qt Creatorのソースコードは zip (26 MB) または tar.gz (21 MB)として入手可能です。 qt.gitorious.org/qt-creator のリポジトリにもアクセスしてみてください。
